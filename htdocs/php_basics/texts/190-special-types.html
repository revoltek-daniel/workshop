Spezielle Typen

In PHP gibt es zusätzlich noch zwei spezielle Typen.

null: Null steht für einen undefinierten leeren Zustand einer Variable. 
<?php
$test; // gültige Definition einer Variable

In diesem Fall ist die Variable null, aber sie ist vorhanden und wir können weiter mit ihr arbeiten. Wenn wir versuchen auf eine undefinierte Variable zuzugreifen wirft PHP eine Notice, welche wir möglichst vermeiden sollten.

ressource:

Immer wenn eine Verbindung hergestellt wird oder auf eine Datei zugegriffen wird haben wir eine Resource

Eine Ressource wird also im Normalfall nur durch eine Funktion einer Variable zugewiesen.
