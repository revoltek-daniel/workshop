Include

Steffis Lieblingsthema. 
<?php
include('header.php');
Die Anweisung include, nimmt eine vorhandene Datei und fügt den Inhalt in die aktuelle Datei ein. Das heißt alle Variablen und Funktionen dieser Datei stehen auch in der aktuellen Datei zur Verfügung. 
<?php
include();
include_once();
require();
require_once();

Der Unterschied zwischen include und require ist, die Art der Fehlermeldung, wenn eine Datei nicht "required" werden kann führt dieses zu einem Ausnahmefehler und die Ausführung wird abgebrochen wohingegen eine include Anweisung nur eine Warning wirft und diese "ignoriert" werden kann.

Das _once sorgt dafür das selbst wenn das include in einer Schleife ausgeführt wird die Datei nur einmal eingebunden wird.

Dynamische Werte an ein include zu übergeben kann gefährlich sein, da so unter Umständen auch auf Dateien zugegriffen werden kann welche nicht erreichbar sein sollen.
