Switch

Eine weitere Kontrollstruktur ist switch.

Wenn man z.B. viele Unterscheidungen hat, die auf dem Wert der selben Variable basieren, kann die Switch Anweisung die richtige wahl sein.

switch ($variable) {
    case konstanterAusdruck1:
        Anweisungen1 // ist optional
    break; // ist optional

    case konstanterAusdruck2:
        Anweisungen2 // ist optional
    break; // ist optional
    ...

    case konstanterAusdruckN:
        AnweisungenN // ist optional
    break; // ist optional

    default: // ist optional
        AnweisungenDefault
}

Fehlt ein break wird der nächste Block mit ausgeführt.
