PHP Funktionen

Um das Rad nicht immer neu erfinden zu müssen, gibt es sehr viele eingebaute PHP Funktionen. Dies ist bestimmt in den letzten Beispielen durch die Autovervollständigung im Editor schon aufgefallen.

Es gibt eine sehr umfangreiche Funktionsreferenz. Interessant für uns, sind vorerst wahrscheinlich die folgenden Seiten:

  • Zeichenketten
  • Mathematische Funktionen
  • Array Funktionen
  • Dateisystem Funktionen
  • Verzeichnis-Funktionen
  • Variablenbehandlung

Keine Angst, diese muss man nicht alle auswendig kennen und viele davon wird man auch wenig bis nie einsetzen.

Beim lesen einer Funktionsdefinition in der Referenz gibt es ein paar Dinge die man wissen sollte

strpos

(PHP 4, PHP 5, PHP 7)

strpos — Sucht die Position des ersten Vorkommens des Suchstrings in einem String
Beschreibung

int strpos ( string $haystack , mixed $needle [, int $offset = 0 ] )

Gibt die numerische Position des ersten Vorkommens von needle innerhalb der Zeichenkette haystack zurück.

Teil Beschreibung
strpos Der Name der Funktion.
(PHP 4, PHP 5, PHP 7) strpos() gab es in allen Versionen von PHP 4 und PHP 5 und PHP 7
int Type des Wertes, den diese Funktion zurückgibt, welche hier ein integer ist.
(string $haystack , mixed $needle [, int $offset = 0 ]) Die Liste an Parametern welche der Funktion übergeben werden. Es wird hier also ein String und ein "mixed" Wert erwartet. Der Wert in Klammern ist optional und im Default Fall 0

Kurz zum "mixed" Typ, diesen hatten wir bisher noch nicht, "mixed" steht einfach dafür das an dieser Stelle verschiedene Typen übergeben werden können.


Für einige dieser Funktionen muss die entsprechende PHP Erweiterung installiert sein. Um herauszufinden welche Erweiterungen installiert sind und auch Einblick in die PHP Konfiguration zu bekommen, gibt es die eingebaute Funktion phpinfo();
