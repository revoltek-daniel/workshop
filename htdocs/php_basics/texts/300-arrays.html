Arrays

Um den letzten Schleifen Typ zu erklären, machen wir kurz einen Ausflug zu einem weiteren Komplexen Datentyp.

Ein Array ist eine Sammlung von Werten, bisher haben wir immer einer Variable einen Wert von einem Datentyp zugewiesen.

<?php
$stringVariable = 'ich bin ein string';
$integerVariable = 123;

Bei einem Array kann man nun mehrere Werte in einer Variable speichern.

<php
$arrayVariable = [
    'stringVariable' => 'ich bin ein string',
    'integerVariable' => 123,
];

Ein Array definiert man entweder mit dem Schlüsselwort array() oder aber über die zu bevorzugende Version über eckige Klammern []

Wie man in dem Beispiel sieht, ist ein Array aus einem Bezeichner oder auch Key und einem Wert aufgebaut, Bezeichner und Wert werden mit => voneinander getrennt.

Der Key ist optional und kann auch eine Zahl sein, ist kein Key angegeben werden die Array Einträge durchnummeriert, beginnend bei 0

Wir können einem Array weitere Werte hinzufügen:

<php
$arrayVariable = [
    'stringVariable' => 'ich bin ein string',
    'integerVariable' => 123,
];
$arrayVariable['key'] = 'neuer Wert';

Der Key ist hier wieder optional und kann auch leer gelassen werden, dann wird wieder ein Numerischer Index vergeben.

Ähnlich wie die Zuweisung der Werte funktioniert auch der Zugriff auf das Array.

<?php
$arrayVariable = [
    'stringVariable' => 'ich bin ein string',
    'integerVariable' => 123,
];

echo $arrayVariable['stringVariable'];
