foreach Schleife

Mit einer foreach Schleife können wir über ein Array iterieren ohne vorher zu ermitteln wie viele Elemente in dem Array sind.

In den meisten Fällen wird man daher diese Schleife verwenden.

<?php
$array = [
   'testwert' => 123,
   'testwert2' => 345,
];

foreach($array as $key => $value) {
    echo 'Der Wert aus dem Eintrag ' . $key . ' ist ' . $value;
}
