for Schleife

Die for Schleife wird normalerweise als Zählschleife benutzt.

<?php
for($i = 0; $i<10; $i++) {
    Anweisung;
}

Wie man im Beispiel sieht, werden im Schleifenkopf, die Zählvariable $i = 0 initialisiert, die Bedingung $i<10 definiert, sowie das inkrementieren der Zähl Variable $i++ implementiert

Bei der for Schleife die als Zählschleife verwendet wird, ist das Risiko eine Endlosschleife zu erstellen geringer, da bei jedem Durchlauf in jedem Fall die Zählvariable erhöht wird.
