Vordefinierte Variablen:

Es gibt in PHP einige vordefinierte Variablen, auch als Superglobals bekannt.

// Werte aus der URL
$_GET

// Werte die in einem Formular "verdeckt" übertragen werden
$_POST

// Werte die in einem Cookie an die Seite übergeben werden.
$_COOKIE

// Dateien die auf den Server übertragen werden
$_FILES

// Alle Daten die an den Server gesendet werden, also auch die Daten aus $_GET und $_POST
$_REQUEST

// Werte die in einer Sitzung für einen User Seitenübergreifend festgehalten werden. Später mehr dazu.
$_SESSION

// Werte die den Server betreffen
$_SERVER

// möglichst nicht benutzen aber recht wichtig im typo3 Kontext. Hier werden Werte in den Globalen Geltungsbereich geschrieben.
$GLOBALS

// Werte die in der Server Umgebung definiert sind.
$_ENV
