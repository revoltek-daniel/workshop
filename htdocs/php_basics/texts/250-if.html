if / else / elseif

Abfrage ob eine Bedingung wahr ist.

Vergleichsoperatoren

  • > Kleiner als
  • < Größer als
  • >= Kleiner gleich
  • <= Größer gleich
  • === gleich, hier werden allerdings auch die Typen der Werte verglichen.
  • !== ungleich mit Typen vergleich
  • == gleich (Kein Typvergleich. Die Verwendung sollte wenn möglich unterbunden werden.)
  • != ungleich (Kein Typvergleich. Die Verwendung sollte wenn möglich unterbunden werden.)
<?php
$name = 'test';
if ($name === 'test') {
    echo 'Name ist test.';
}

Mehrere Bedingungen können miteinander kombiniert werden, dafür können Bedingungen mit && oder || verknüpft werden.

&& steht hier für UND, || steht hier für ODER.

<?php
$wert = 10;
if ($wert > 5 && $wert < 15) {
    echo $wert;
}

Durch Klammerung können komplexe Bedingungen realisiert werden

<?php
$wert = 10;
if ($wert != 11 && ($wert > 5 && $wert < 15)) {
    echo $wert;
}
