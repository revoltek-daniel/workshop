Kurz Ausflug: Was ist eine Scriptsprache:

Skriptsprachen werden nicht in Byte code umgewandelt, sondern direkt beim Ausführen übersetzt.

PHP wird auf dem Server ausgeführt (es gibt Möglichkeiten mehr oder weniger Desktop Apps draus zu machen aber das am besten direkt wieder vergessen)

PHP liegt aktuell in der Version 7.3 vor. PHP 6 gibt es nicht und wurde übersprungen.
