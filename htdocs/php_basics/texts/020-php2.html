PHP wird IMMER in .php Dateien gespeichert, alle anderen Dateiendungen sollten nicht benutzt werden.

Ein PHP Script beginnt immer mit einem öffnenden <?php Tag, alles danach wird durch den Interpreter ausgeführt.

Es gibt die Möglichkeit HTML und PHP in eine Datei einzubauen, allerdings ist dies spätestens in komplexeren Projekten nicht mehr gewünscht. Deswegen sollte man auf eine Trennung achten.
Innerhalb dieses Workshops werden wir allerdings auch mal beides mischen.
