Allgemeines zur Syntax

Im Normalfall hat man eine Anweisung pro Zeile, dies ist zwar kein muss aber sollte man für die Übersichtlichkeit beachten.

Jede Anweisung muss mit einem ; beendet werden

Um den Code lesbar zu gestalten sollte auf eine ordentliche Einrückung geachtet werden

<?php
echo 123; echo 456; // möglich aber nicht unbedingt gewünscht

echo 123;
echo 456; // schöner :)
